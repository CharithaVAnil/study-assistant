# Smart Study Assistant

A simple web application to help students organize study tasks and manage study sessions.

## Features
- Add study tasks with deadlines and priority
- Generate a study timetable
- Mark tasks as completed
- 25-minute Pomodoro study timer
- Tasks saved using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript
